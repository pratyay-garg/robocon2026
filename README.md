## INSTRUCTIONS TO RUN THE PROJECT

---

requirements.txt file contains the modules need to be installed to run the files.

train.py makes the model and saves it into (robocon_model.pth) file.

to train the model (for blue team, edit train_red and val_red to train_blue and val_blue in train.py file)

export_onnx.py file converts it into onnx format.

inference.py will run the model.
